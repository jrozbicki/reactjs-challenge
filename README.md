### To get started

1. Clone repo and change directory

   ```
   git clone https://github.com/jrozbicki/reactjs-challenge.git
   cd reactjs-challenge
   ```

2. Install dependencies

   ```
   npm install
   ```

3. Start client server (port 5000)

   ```
   npm run server
   ```

4. Start client server (port 3000)

   ```
   npm start
   ```

5. Run unit tests

```
npm run test
```
